# GitHub Tools

## create-pr
Cria um Pull Request no GitHub.
Parametros: base, head, title, body

## list-prs
Lista Pull Requests abertos.
Parametros: state (open/closed/merged)

## run-workflow
Executa um workflow do GitHub Actions.
Parametros: workflow, branch

## list-workflows
Lista workflows disponiveis.
Parametros: (nenhum)
