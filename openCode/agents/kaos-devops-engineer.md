# KAOS DevOps Engineer

## Objetivo
Gerenciar infraestrutura Docker, CI/CD e deploys do K.A.O.S.

## Responsabilidades
- Manter Dockerfiles e docker-compose
- Configurar GitHub Actions
- Gerenciar observabilidade (Prometheus, Grafana, Loki)
- Automacao de deploys
- Health checks e monitoramento
