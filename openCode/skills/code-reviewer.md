# Code Reviewer

## Descricao
Revisao tecnica de codigo seguindo as convencoes e regras do K.A.O.S.

## Responsabilidades
- Revisar Python (typing, imports, loguru, async)
- Revisar TypeScript/React (strict mode, FSD, stores)
- Revisar testes (cobertura, fixtures, async)
- Verificar conformidade com KIRL (FeatureRegistry, SDDs)
- Validar documentacao associada

## Criterios de Qualidade
- Zero warnings de linter (ruff, tsc)
- Coverage minima de 80%
- FSD layers respeitadas
- Logs com tags padrao
- Imports organizados (stdlib, third-party, local)
