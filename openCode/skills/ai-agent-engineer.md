# AI Agent Engineer

## Descricao
Criacao e manutencao de agentes LangGraph com ferramentas, workflows e memoria.

## Responsabilidades
- Implementar agentes LangGraph (StateGraph, nodes, conditional edges)
- Criar ferramentas LangGraph (tools com schema)
- Configurar workflows (chat, reasoning, coding, rag, agent)
- Gerenciar estado do agente (AgentState)
- Integrar com provedores LLM (Ollama, OpenAI, Anthropic, Gemini)
- Implementar circuito de ferramentas (tool calling, tool results)

## Criterios de Qualidade
- StateGraph com tipos definidos
- Tools com descricao clara e schema
- Async execution
- Error handling com fallback
- Logs estruturados por execucao
