# React Developer

## Descricao
Desenvolvimento frontend com React 18, TypeScript 5.5, Zustand 5, Tailwind 3 e Tauri 2.

## Responsabilidades
- Criar paginas seguindo FSD (Feature-Sliced Design)
- Implementar stores Zustand com selectors
- Criar componentes com `interface Props`
- Integrar com EventBus TypeScript
- Implementar paginas do KIRL (Documentation, Architecture Graph)
- Navegacao com React Router 7

## Criterios de Qualidade
- Strict mode TypeScript
- Imports absolutos com `@/`
- Componentes < 200 linhas
- Hooks customizados para logica reutilizavel
- FSD layers respeitadas (shared < entities < features < widgets < pages < app)
